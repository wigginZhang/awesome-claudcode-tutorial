你有没有发现，现在网上教 Claude Code 的内容，要么是官方文档的翻译，要么就是几个命令的罗列？看完之后还是不知道怎么用。

今天给你推荐一个项目——「Claude Code 从入门到精通」，这是目前 GitHub 上最全面的 Claude Code 中文教程，没有之一。

它最特别的地方是提供了两种学习方式：

第一种是文章式学习，200 多篇详细文章，按主题分类，想查什么直接搜。

第二种是课程式学习，29 个章节，从安装配置到企业级部署，一条线带你从头学到尾。

内容覆盖也很全：

前几章教你安装、登录、做第一个项目，完全零基础友好，15 分钟就能做出一个待办清单网页。

中间章节讲权限管理、版本回滚、上下文管理这些实际开发中每天都会用到的功能。

后面还有企业级内容——CI/CD 集成、监控可观测性、多 Agent 协作、Computer Use 屏幕操作，这些你在别的地方基本找不到。
